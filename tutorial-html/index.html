Hyper Text Markup language atau biasa disingkat HTML adalah sebuah bahasa pemograman untuk menghasilkan dokomen-dokumen hypertext yang digunakan didalam World Wide Web (WWW). HTML terlihat sebagai pemograman lama, dimana kode-kode html diawali dan diakhiri untuk menunjukan keluaran saat dijalankan di sebuah web browser.
