Background Color

Property CSS background berguna untuk menentukan Efek latar belakang pada suatu elemen HTML. Property yang biasa digunakan dalam CSS Background antara lain :

  • latar belakang warna yaitu dengan background-color atau background saja
  • latar belakang gambar yaitu dengan background-image

Sekarang Anda akan membuat background color pada suatu element HTML. Pada contoh dibawah ini, ubah warna latar belakang dari css selektor #latarbelakang menjadi hijau dengan menambahkan properti css dengan nama properti background dan value properti green.
Contoh : #selektor{property:value}

Selanjutnya membuat background image, atur element #backgroundgambar memiliki background berupa gambar ini. Caranya adalah dengan menambahkan kode properti background-image : url("lokasi-background")
