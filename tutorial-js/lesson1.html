JavaScript Pertama Kita

JavaScript dapat "display" data dengan berbagai cara:

  • Menggunakan alert box, menggunakan: window.alert()
  • Menggunakan HTML sebagai output menggunakan: document.write()
  • Menggunakan HTML element, menggunakan: innerHTML

Kamu dapat menggunakan cara alert box: Contoh : alert("value"); / alert("abc");

Using window.alert()

JavaScript can "display" data in different ways:

  • Writing into an alert box, using window.alert()
  • Writing into the HTML output using: document.write()
  • Writing into an HTML element, using innerHTML
  • Writing into the browser console, using: console.log()
